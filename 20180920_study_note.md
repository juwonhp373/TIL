# 라이프사이클(= 생명주기)

**라이프 사이클**
: : 인스턴스의 상태에 따라서 호출 할 수 있는 속성들이다.

**라이프 사이클 훅** 
: :  라이프 사이클의 속성마다 **개발자가 추가한 커스텀 로직** 이다.

뷰 인스턴스도 객체이기 때문에 생명주기를 가진다. 

**속성의 종류 8가지**
: beforeCreate,
created,
beforeMount,
mounted,
beforeUpdate,
updated,
beforeDestroy,
destroyed
